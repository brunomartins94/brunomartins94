## 🚀 Projeto em destaque

### 🗳️ [Observatório Eleitoral 2026](https://github.com/brunomartins94/observatorio-eleitoral-2026)

Pipeline de dados end-to-end com dados públicos do TSE brasileiro.

- **1.044.734 candidatos** processados (2018, 2022 e 2024)
- **11 milhões de registros** — candidaturas, receitas e despesas
- Stack: Airflow · MinIO · DuckDB · dbt · Streamlit · Docker · GitHub Actions

> *"A representatividade feminina cresceu de 31,5% para 34,3% entre 2018 e 2024."*
